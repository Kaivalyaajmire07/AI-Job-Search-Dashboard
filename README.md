# 🚀 AI-Powered Job Search Dashboard

An intelligent Job Search Dashboard built using **Python**, **Streamlit**, **Artificial Intelligence (AI)**, and **Natural Language Processing (NLP)**. The application searches real-time job listings from the **JSearch API (RapidAPI)** and provides AI-powered ranking, recommendation, and filtering to help users find the most relevant opportunities.

---

## 📌 Features

- 🔍 Real-time Job Search using JSearch API
- 🤖 AI-based Job Ranking
- 🧠 NLP Query Expansion
- 🎯 AI Job Recommendation Engine
- 🛠️ Skill Extraction from Job Descriptions
- 📍 Multiple Location Search
- 🌍 Country-wise Job Search
- 💼 Employment Type Filter
- 🎓 Experience Level Filter
- 📅 Posted Within Filter
- 🌐 Remote Job Filter
- 💰 Minimum Salary Filter
- 🚫 Duplicate Job Detection
- ⭐ Match Score & Recommendation Score
- 📄 Pagination Support
- ⚡ Modern Streamlit UI
- 🔐 Secure API Key Management using `.env`

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Kaivalyaajmire07/AI-Powered-Job-Search-Dashboard.git
cd AI-Powered-Job-Search-Dashboard
```

### Create Virtual Environment

```bash
python -m venv job
```

### Activate Environment

Linux / macOS

```bash
source job/bin/activate
```

Windows

```bash
job\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root.

```env
RAPIDAPI_HOST=jsearch.p.rapidapi.com

RAPIDAPI_KEY1=YOUR_API_KEY_1
RAPIDAPI_KEY2=YOUR_API_KEY_2
RAPIDAPI_KEY3=YOUR_API_KEY_3
RAPIDAPI_KEY4=YOUR_API_KEY_4
RAPIDAPI_KEY5=YOUR_API_KEY_5
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```
